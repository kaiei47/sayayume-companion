# ストーリーモード テスト仕様書

## 1. ストーリー選択ページ（/story）

### 1.1 ヘッダー
- [ ] ロゴ（SayayumeLogo）が表示される
- [ ] 「ストーリー」タイトルが中央に表示される
- [ ] ロゴクリックでホームに遷移

### 1.2 親密度表示
- [ ] さや/ゆめの親密度レベルが正しく表示（10段階）
- [ ] 色分け（さや=ピンク、ゆめ=パープル）

### 1.3 フィルター
- [ ] 「すべて」「さや」「ゆめ」「2人」ボタンが表示
- [ ] 各フィルターで正しくストーリーが絞り込まれる
- [ ] アクティブフィルターがピンクでハイライト

### 1.4 ストーリーカード
- [ ] サムネイル絵文字が表示される
- [ ] タイトルが太字で表示
- [ ] キャラタグの背景色が正しい（さや=ピンク背景、ゆめ=パープル背景、duo=アンバー背景）
- [ ] 難易度★が正しく表示（★1なら★☆☆☆☆）
- [ ] プレイ時間（15分）が表示
- [ ] 説明文が表示
- [ ] ロック中のストーリーは鍵アイコン+薄く表示+ロック理由表示
- [ ] クリア済みストーリーは「クリア済み」バッジ+緑ボーダー

### 1.5 セッション状態
- [ ] 未プレイ: 「はじめる」ボタン
- [ ] プレイ中: 「続きから」+「最初から」ボタン+ミッション進捗バー
- [ ] クリア済み: 「もう一度プレイ」ボタン

### 1.6 ボトムナビ
- [ ] ホーム/チャット/ストーリー/設定の4タブ
- [ ] 「ストーリー」がアクティブ（ピンク）
- [ ] 各リンク先が正しい

## 2. ストーリープレイページ（/story/[storyId]）

### 2.1 ヘッダー
- [ ] 「← 戻る」リンクが機能する
- [ ] ストーリータイトル+サムネイルが表示
- [ ] ミッションカウント（0/3）が表示
- [ ] ミッション進捗バーが表示
- [ ] ミッションパネルトグルボタン（📋）が機能する

### 2.2 ミッションパネル
- [ ] 開閉できる
- [ ] 各ミッションの名前が表示
- [ ] 未達成は☐、達成済みは✅
- [ ] ヒントが表示される

### 2.3 チャットUI
- [ ] 初期表示: ストーリー説明+「メッセージを送ってストーリーを始めよう」
- [ ] ユーザーメッセージ: 右寄せ、ピンク背景
- [ ] AIメッセージ: 左寄せ、暗い背景
- [ ] ストリーミング表示（タイプアニメーション）
- [ ] 自動スクロール

### 2.4 ミッション達成
- [ ] ミッション達成時にアニメーション（バウンス）
- [ ] ミッション名が表示される
- [ ] 進捗バーが更新される
- [ ] ミッションパネルのチェックが更新される

### 2.5 ストーリー完了
- [ ] 全ミッション達成でクリアモーダル表示
- [ ] 称号が表示される
- [ ] 親密度ボーナスが表示される
- [ ] 「ストーリー一覧」「チャットへ」ボタンが機能

### 2.6 入力エリア
- [ ] テキスト入力できる
- [ ] Enter送信、Shift+Enter改行
- [ ] ストリーミング中は送信無効
- [ ] クリア後は入力無効

## 3. API動作

### 3.1 /api/story/sessions GET
- [ ] ログイン済みユーザーのみアクセス可
- [ ] ストーリー一覧+セッション情報を返す
- [ ] プラン別の難易度制限が反映される

### 3.2 /api/story/sessions POST
- [ ] 新セッション作成成功
- [ ] 既存in_progressセッションはabandonedに

### 3.3 /api/story/chat POST
- [ ] SSEストリーミングが正常動作
- [ ] 親密度更新される
- [ ] ミッション判定が動作する
- [ ] 完了時にパーソナルメモリ保存

## 4. 発見済みバグ（スクリーンショットから）

- [ ] BUG-1: キャラタグの背景色が正しく表示されない可能性（Tailwind動的クラスパージ問題）
- [ ] BUG-2: ボトムナビの絵文字アイコンがOSによって見た目が異なる可能性
- [ ] BUG-3: 親密度表示が枠だけでグラデーション背景が薄い
