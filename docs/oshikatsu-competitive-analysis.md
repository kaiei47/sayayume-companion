### 推し活で使うお金TOP5（リアルアイドルファン）

2. グッズ・マーチャンダイズ

5. ファンクラブ会費

### 40-50代男性の推し活特徴（さやゆめのTikTok視聴者層）
- **月額支出は「5千円未満」が最多** — 若年層より低額だが安定的

- **テレビ・SNSで偶然見かけて始める** — 受動的なきっかけが多い
- **チケット代・イベント費用に最もお金を惜しまない**
- **「AIという知的アリバイ」でエンゲージしやすい** — 生身のアイドルより心理的ハードルが低い
- **推し活は40-50代にとって「精神的充足」と「アイデンティティ形成」の機能がある**

## 7. さやゆめに実装可能な推し活機能（優先度付き）

### Phase 1: 即実装可能（チャット+画像生成+Webアプリのみで完結）

#### 7-1. デジタルチェキ機能 ⭐⭐⭐ 最重要
**概要**: さや/ゆめとの「記念写真」をAI画像生成で作成
- ユーザーが衣装・ポーズ・シチュエーションを選択
- 画像にフレーム・日付・サイン風テキストをオーバーレイ
- **「マイチェキ帳」**にコレクション保存
- 1日1枚無料 / 追加はトークン消費
- **レアチェキ**（水着・ドレス等）は有料限定

**なぜ刺さるか**: チェキは地下アイドルの収益の9割。「推しとの1対1の記念」に課金する心理は実証済み。

#### 7-2. 誕生日イベント ⭐⭐⭐
**さや/ゆめの誕生日**:
- 特別チャットシナリオ（誕生日モード）
- 限定誕生日チェキ（期間限定）
- プレゼント選択（デジタルギフト）→ 反応が変わる
- バースデーケーキ画像生成

**ユーザーの誕生日**:
- 登録した誕生日にプッシュ通知で「おめでとう！」
- さや/ゆめからの特別メッセージ（音声付きなら最高）
- 誕生日限定チェキ（ユーザーの名前入り）
- **無料ユーザーにも全開放** → 有料転換のきっかけに

#### 7-3. おでかけ/デートモード ⭐⭐⭐
**OSHIAI「おでかけモード」の進化版**:
- ロケーション選択: カフェ / 水族館 / 遊園地 / 花火大会 / 学校 / 海 etc.
- 時間帯連動: 朝・昼・夕方・夜で雰囲気が変わる
- **デート中に画像生成**: そのロケーションでのさや/ゆめの画像を生成
- 各ロケーションに固有のトーク内容
- Free: 3ロケーション / Basic: 10 / Premium: 全開放
